# 🌦️ Global Weather API Data Analysis Project

## Overview
An end-to-end data analysis project that fetches live weather data 
from 15 cities (10 Indian + 5 global) using the OpenWeatherMap API, 
cleans and transforms it with Pandas, and extracts insights through 
exploratory data analysis and visualization.

## Skills Demonstrated
- REST API integration using Python `requests`
- Data extraction, cleaning & transformation with `Pandas`
- Exploratory Data Analysis (EDA)
- Custom metric creation (Comfort Index)
- Data visualization with `Matplotlib` & `Seaborn`
- Error handling & production-style Python scripting

## Key Insights Found
- Indian cities averaged 10°C hotter than global cities in April
- Humidity showed a 0.42 correlation with temperature
- Bangalore ranked as the most comfortable Indian city
- Clear skies dominated Rajasthan cities while coastal cities showed haze


## How to Run
```bash
pip install requests pandas matplotlib seaborn
python phase2_multi_city.py    # fetch data
python phase3_pandas_csv.py    # clean & save CSV
python phase4_analysis.py      # analyze
python phase5_visualizations.py # generate charts
```
